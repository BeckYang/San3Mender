# 三國志3存檔修改器
支援三國志3 dos 中文版及日文版

## Pre-request 使用前需求
1.取得 Java 執行階段環境 (JRE/JDK) 版本 8 以上(版本 8/11/17 測試過沒問題，其他版本應該也可以)
2.從 Release 取得編譯好的 Jar 檔案及相依檔案 或 自行下載原始碼透過 Maven 編譯


## 使用方式
透過命令列執行 `javaw -jar San3Mender-1.0.jar` 即可啟動修改器使用介面

基本操作流程
 - "存檔位置" 先選擇檔案 SANGOKU3.SAV，再選擇要改哪一組紀錄
 - 選擇目標(武將或城市...)進行修改，按"套用變更"將當下選擇(武將或城市...)回存到記憶體
 - 改好其它目標後，最後按"儲存"才會真正寫回檔案(在儲存之前切換紀錄，將會丟棄當下的修改!)

本修改器依照遊戲程式規則將部分欄位設限，以避免資料面出現問題，
因此使用介面上允許編輯的欄位都支援修改，本修改器另支援以下修改:
1. 讓武將在進行間諜工作: "在"工作(!)" 標題文字雙擊滑鼠來啟用，再依提示操作。
2. 讓後期武將提早登場: 首先找到目標武將，修改其"所在都市"，
  再依照提示選擇即可，建議一併修改"生年"，以免年齡顯示不正確
  注意: 每年1月是武將登場時間，若劇本內輪到該武將登場，遊戲仍會強制修改該武將狀態並移動到登場城市!


## 關於三國志3 dos 中文版的新武將人名
中文版的登場武將人名使用自帶字庫有固定編碼，本修改器已內建大部分漢字對應表，
但是新武將人名的漢字使用造字編碼，字型似乎儲存於檔案內不同位置，
在沒有對應表的情況下，本修改器會顯示該造字之16進位碼(由數字0~9及A~F組成)，
若要讓本修改器顯示該文字，可於"設定"頁籤內自行建立對應表，
參照範例如下: (每一行之格式為: 16進位字元4碼 + '=' + 對應的一個字)
D98E=娜
D9CA=橘


## FAQ
1.有辦法阻止武將老死或病死嗎?
這類事件發生前(似乎是每年1月)，通常"近期死亡"會被勾選，只要取消勾選並存檔，就能阻止武將自然死亡。

2.如何復活死亡的武將?
武將死亡時，程式會將"生年"改成0，把生年重新填上並再按"套用變更"會重新詢問其新狀態。
依照上述規則，如果想登用流浪的君主，最好先讓他們死亡，然後再復活以免造成問題。

3.流浪君主何時會佔據空白都市?
流浪君主的兵力達一定數量(兵力較周邊州郡多?)，兵糧足夠(兵力至少比周圍城市多)就可能再立旗。
利用本修改器 "外交/流浪" 頁籤進行修改可簡單讓它們重新立旗。

4.為何人物或都市的序號出現*?
修改人物或都市資料，並"套用變更"後，會將該筆資料加上*標示，存檔後就會清除該標示。
(修改武將"所在都市"或更動順序，可能連帶修改其他武將相關資料，此時其他武將也會有*標示)
